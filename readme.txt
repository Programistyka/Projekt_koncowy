MoneyHelper to aplikacja internetowa stworzona w Django, zaprojektowana, aby pomóc skutecznie zarządzać finansami. 

Funkcje:  

Dodawaj, usuwaj i edytuj dochody i wydatki.  

Filtrowanie dochodów i wydatków według zakresu dat i kategorii.  

Obliczanie sumy dochodów i wydatków w wybranym okresie. *  

Obliczanie Oszczędności  dla wybranego zakresu dat. * 

Dodawaj, usuwaj i edytuj przypomnienia.  

Sortowanie przypomnień według daty  

Technologie wykorzystane  

Python 

Django 

SQLite 

Bootstrap
